JavAct: A Java(TM) library for distributed and mobile actor-based computing
Version : 1.5.3
Date : 07/2007
Equipe SMAC, IRIT (ex. IAM et LYRE)

INSTALLATION : L'archive jar de distribution de JavAct est auto-exécutable, elle contient le programme d'installation. Sous Windows un double-click sur l'archive permet de l'activer. En ligne de commande (tous OS) : java -jar JavActv153.jar
	
Pour plus d'informations, reportez-vous au tutoriel de JavAct :
    http://javact.org 
    
LANCEMENT :
Le répertoire "bin" contient un ensemble de scripts pour simplifier/automatiser le lancement de JavAct. En particulier le script "javactvm" permet de lancer le système d'accueil nécessaire au fonctionnement de JavAct en mode réparti.
Nous vous conseillons pour le développement et les tests l'utilisation du plugin JavAct.
